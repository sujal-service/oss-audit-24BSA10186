## Project Overview
This repository contains the practical shell scripting component for the OSS Capstone Project. The scripts demonstrate Linux system administration, log analysis, and automation.

## Script Descriptions
1. **system_identity.sh:** Displays OS information and user details.
2. **foss_inspector.sh:** Checks the installation and version of the selected FOSS.
3. **disk_auditor.sh:** Analyzes directory sizes and permissions.
4. **log_analyzer.sh:** Counts occurrences of keywords in system logs.
5. **manifesto_gen.sh:** Generates a personalized Open Source manifesto.

## How to Run
1. Give execution permission: `chmod +x *.sh`
2. Execute a script: `./script_name.sh`
